# Week Weather iOS App 🌤️

A simple SwiftUI-based iOS application that displays weekly weather forecasts in a clean and structured UI.

## 🚀 Features
- Weekly weather display (Mon–Sun)
- Clean and minimal UI
- Reusable SwiftUI components
- Structured layout using VStack & HStack

## 🛠 Tech Stack
- Swift
- SwiftUI
- Xcode
